## Booking System

### Session Types Available
1. **Personal Session** - One-on-one private session with our practitioner
2. **Couple Session** - Shared healing experience for two people
3. **Group Session** - Experience healing with a small group (3-6 people)

### Session Durations
- **30 Minutes** - $49 (Quick healing session)
- **45 Minutes** - $69 (Extended session)
- **60 Minutes** - $89 (Full healing experience - Group sessions only)

### Booking Process
The booking system follows a simple 3-step process:
1. **Select Session Type** - Choose between Personal, Couple, or Group
2. **Choose Duration** - Select your preferred session length
3. **Enter Contact Information** - Provide your name, email, and phone number

## Getting Started

### Prerequisites

- Modern web browser
- Node.js (optional, for local development)

### Installation

1. Clone the repository
```bash
git clone https://github.com/ronikakamdar/soundhealing_with_RSD.git
cd soundhealing_with_RSD
```

2. Open in browser (no installation needed for basic use)
```bash
# Simply open index.html in your browser
# OR use a simple HTTP server
python -m http.server 8000
# Then visit http://localhost:8000
```

3. For development with live server
```bash
npm install
npm start
```
